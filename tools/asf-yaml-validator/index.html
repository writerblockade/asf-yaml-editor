Offline .asf.yaml Validator & Builder

Create, edit, and validate an Apache .asf.yaml · runs in your browser; the only network call is the optional Import from a repo fetch you trigger · schema snapshot: —

Not a guarantee of acceptance. This is an independent, unofficial re-implementation of the checks in Apache's public .asf.yaml schema — which is itself a work in progress — not the server-side code ASF runs to apply your config. It reads and reports only. By default it makes no network calls; the one exception is the opt-in Import from a repo button, which does a read-only GET of a public .asf.yaml you ask for and nothing else. It never writes to, commits to, or changes your repo. Treat a clean result as "no obvious problems," not "guaranteed valid." When in doubt, test on a non-default branch first.
